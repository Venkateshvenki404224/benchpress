# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| `develop` (latest) | Yes |
| Older branches | No |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please report them via email:

**Email:** venkateshvenki404224@gmail.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment:** Within 48 hours
- **Initial assessment:** Within 5 business days
- **Fix timeline:** Depends on severity, but we aim for patches within 2 weeks for critical issues

## Scope

This policy covers the BenchPress Frappe app, including:
- API endpoints (`benchpress/api.py`)
- Docker container management
- WireGuard VPN configuration
- Frontend authentication and authorization
